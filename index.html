用hugo搭建个人博客

Step 1：下载 Hugo Mac用户 brew install hugo hugo version windows用户 下载地址 Step 2：创建一个新的站点 hugo new site quickstart Step 3：添加一个主题 cd quickstart git init git submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke Step 4:添加一片文章 hugo new posts/my-first-post.md 进入该文件就可以开始写自己的博客啦,md文件后会看到开头有一些信息，title是博客的标题，draft: true是表示当时博客是以草稿的形式，需要将他改为draft: true。 --- title: "Test" date: 2020-06-29T22:46:12-07:00 draft: true --- Step 5：开启hugo服务器 hugo server -D 服务器运行后会看到本机的一个地址，点击进入即可看到博客内容 Step 6：制定主题 打开config.toml对其更改 Step 7：构建一个静态页面 hugo -D Step 8：部署 在hugo站点根目录下创建`.gitignore`写入/public/，目的是让Git仓库不要管public文件。 在GitHub上新建仓库名为xxx.github.io(xxx表示你自己的github名称)。接着进入public文件初始化git仓库。 链接远程仓库也就是对应刚刚创建的仓库，并将所有的文件提交到本地仓库，再push到远程仓库。 进入刚刚创建的GitHub仓库，点击settings找到GitHub pages如下图将其改为master branch选项。 官方教程
